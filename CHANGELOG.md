# 更新日志

本文件记录每次发布的版本与更新内容，**最新版本放在最上方**。
每完成一个需求，就在最新版本条目下加一行简短说明（一句话即可）。

格式约定（发布脚本 `backend/deploy/release.sh` 依赖它解析）：

- 每个版本一个 `## 版本号` 标题，版本号形如 `0.2.0+2`（`x.y.z` 是展示版本名，
  `+N` 是 Android versionCode，必须递增）。
- 标题下用 `-` 列出本次更新内容，每条一句话。
- 顶部第一个 `## ` 条目即「待发布/最新」版本，发布脚本只读它。
- 顶部版本号必须与 `pubspec.yaml` 的 `version:` 一致（脚本会校验，不一致会拒绝发布）。

## 0.1.0+6

- 新增「囤货铺」插件：记录家庭囤货与共享采买清单两个联动列表；囤货可加减数量、设低量阈值，见底时一键补进采买，采买买到后还能顺手入库；首页卡片在有东西要补货时提醒，平时显示采买清单剩几项

## 0.1.0+5

- 回忆：点击图片可全屏看大图，支持左右滑动浏览上一张 / 下一张（视频则点击播放）
- 回忆：时间线、详情与留言里显示创建者的真实头像（未设头像时回退到 emoji 头像）
- 记账：账单列表里显示记录人的真实头像（未设头像时回退到 emoji 头像）
- 菜谱：详情页「由谁添加」显示作者真实头像（未设头像时回退到 emoji 头像）
- 家务：列表负责人与指派选择器显示真实头像（未设头像时回退到 emoji 头像）
- 家庭管理：成员列表与角色操作里显示成员的真实头像（未设头像时回退到 emoji 头像）

## 0.1.0+4

- 新增「回忆」插件：以贯穿的时间线记录生活，可上传照片或短视频、写标题和文案，另一半还能留言；首页卡片展示最近一段回忆的标题与总条数
- 回忆：新建时上传的第一张照片会自动读取拍摄日期填入「日期」，之后仍可手动修改（编辑已有回忆不触发）
- 应用内更新：下载过程中离开「关于」页不再中断下载，重进页面可继续看到下载进度
- 通知偏好：切换某个开关时，其它开关与总开关不再闪一下变灰
